## 真实案例

### 案例1："国能App" — 冒用央企，18万人被骗20亿（2026年）

诈骗团伙冒用"国家能源集团"名义，伪造"国能App"理财平台，以"国企背书、零风险稳赚"为诱饵。服务器架设境外，盗用官网版式，虚构煤矿投资项目，租酒店开宣讲会，穿仿制工装。**18万受害者，20亿元资金**被卷走。

### 案例2："全民养牛" — 云养牛集资诈骗5.6亿（2026年4月宣判）

搭建"全民养牛"App，号称5000元认养澳洲肉牛、奶牛，承诺5%–15%年化收益。平台无任何真实存栏牛只，视频均为网络截取。**涉案5.6亿余元**，数万投资者受损。主犯张某某被判**有期徒刑10年6个月**，罚金75万元。

### 案例3："君河资产" — 美女荐股+10倍杠杆跑路（2026年3月）

诈骗团伙在小红书、抖音以"美女"身份主动添加股民，推荐股票建立信任，诱导下载"君河资产"App。提供**10倍杠杆**交易，宣称"交易所姑妈"提供内幕消息。2025年7月–12月运营，12月3日突然跑路。**超百人受害，涉案金额过亿**。

### 案例4："太白投资" — 二次套壳骗局（2026年2月）

碰瓷淡马锡旗下太白投资，**涉案超5亿元**，全国超20个省份上千人受害。受害人多被"网恋"套路。第一次跑路后改名再次行骗。

### 案例5：广州AI投资 — 50万打水漂（2026年5月）

骗子伪装成证券技术员，谎称"AI智能投资技术"可精准测算股市黄金点位，诱导许女士登录虚假投资网站。投资50万元，账面显示盈利150万元后平台无法登录。警方追回20余万元。

### 案例6：量派投资"李鬼" — 高仿私募平台（2026年1月）

冒充百亿级私募"量派投资"，用空壳公司的"脉达App"锁定目标。套路：免费体验20日+亏损包赔→诱导下载"量派阿尔法"App→允许小额提现→大额时"系统故障"。

## 一般模式

```
第一步：包装
  │  冒用国企/知名金融机构名义
  │  伪造官网、App、营业执照
  │  租用高档写字楼开宣讲会
  ▼
第二步：引流
  │  美女/帅哥社交平台主动私信
  │  短视频/公众号投放理财广告
  │  炒股群、投资群内"老师"带单
  ▼
第三步：养鱼
  │  小额投入 → 准时返利 → 允许提现
  │  制造"稳赚不赔"的心理暗示
  │  群内"托儿"不断晒收益截图
  ▼
第四步：收割
  │  诱导大额投入 → 平台维护/系统故障
  │  提现需缴"保证金""税费""解冻费"
  │  → 最终跑路，平台关闭
```

## 警示

### ⚠️ 识别虚假投资平台的5个特征

1. **收益高得离谱**——承诺年化10%以上甚至"日收益1%"，远超正常投资回报
2. **保本保息**——正规投资均有风险，"保本保息""稳赚不赔"是违法表述
3. **向个人账户转账**——正规投资走第三方存管，绝不转给个人银行卡
4. **非官方渠道下载App**——通过链接/二维码下载的App大概率是诈骗
5. **小程序/网页版为主**——虚假平台不敢上架官方应用商店

### ⚠️ 新概念包装需警惕

`AI炒股` `量化交易` `云养殖` `元宇宙` `央行数字人民币` `碳汇` `新能源`——都是诈骗分子热衷包装的热门概念。

### ⚠️ 防骗原则

- **不下载不明链接的App**——通过官方应用商店下载
- **不相信"内幕消息"**——真有内幕不会告诉你
- **不向私人账户转账**——认准正规托管账户
- **收益过高必有诈**——你想的是利息，骗子要的是本金
- **遇到可疑投资，拨打96110咨询**

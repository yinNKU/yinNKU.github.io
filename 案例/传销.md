## 真实案例

### 案例1：长春"文创盲盒" — 日流水150万的传销盘（2025年12月宣判）

打着数字藏品、文创产品旗号，花3000元买看不见的盲盒，每天加价4%转手，发展下线可获推荐奖。盲盒里实际是廉价茶叶或手链。下线层级达10级，直接/间接发展会员**3397人**，日流水突破**150万元**。主犯陈某被判有期徒刑2年，罚金10万元；其余6名骨干获刑6个月至1年6个月。

### 案例2：西安"1040阳光工程" — 320名警力收网抓获130人（2026年6月）

"交69800元，发展3个下线，2–3年赚1040万"——纯资本运作，无任何实体产品。西安警方出动320名警力，清查传销场所180余处，抓获**130人**。4名核心骨干采取刑事强制措施，27人被治安拘留。

### 案例3：内江"圆梦茶海" — 虚拟种茶7万会员（2023年收网，2026年报道）

手机App虚拟"种茶"，宣称"三年赚百万元"。买168元白酒得"门票"→购买虚拟土地→种植虚拟茶树。用"二叉树"App架构逃避打击，以"国学+实体+成功学"包装，租用真实茶园制造假象。吸纳**7万多名**会员。主犯徐某某被判**6年**，罚金60万元。

### 案例4：宁波"红韵串商" — 社交电商资金盘，年化收益292%（2026年5月立案）

以"社交电商"为幌子，搭建"积分商城"，交99元激活费，充值最高35万元，年化收益宣称达**292%**。多层级发展下线，"静态返利+动态奖励"。

### 案例5：湖北咸宁"凤凰潮" — 专门针对老人的传销（2026年6月立案）

以"零创赚钱""助农振兴"为幌子，充值购买会员，设置多级代理层级。以"平台升级""政策调整"为由关闭提现，老人被骗金额4000–20000元不等。平台发行的"凤凰币""工分"均为平台操控的虚拟资产，无任何真实价值。

## 一般模式

```
第一步：概念包装
  │  数字藏品 / 文创盲盒 / 社交电商
  │  虚拟种茶 / 养老投资 / 区块链
  │  用热门概念掩盖传销本质
  ▼
第二步：高利诱惑
  │  "日收益4%""年化292%"
  │  "三年赚百万""2-3年赚1040万"
  │  → 远超正常投资回报率
  ▼
第三步：拉人头
  │  发展下线可获得推荐奖、团队奖
  │  层级越深，收益越高
  │  → 金字塔结构
  ▼
第四步：收割跑路
  │  资金池无法维持时关闭提现
  │  "系统升级""政策调整"
  │  → 平台跑路，血本无归
```

## 受众

- **有投资欲望但缺乏专业知识的人**
- **老年人**（被"养老投资""助农"等名义吸引）
- **想赚快钱的人**（被高额回报诱惑）
- **失业、无业者**（被"零创赚钱"吸引）

## 警示

### ⚠️ 传销的三大核心特征（万变不离其宗）

1. **交入门费** — 必须先投入资金才能获得资格
2. **拉人头** — 靠发展下线来获取收益
3. **团队计酬** — 按层级进行抽成和奖励

> 不管包装成什么概念——盲盒、数字藏品、社交电商、虚拟种茶——只要同时具备以上三个特征，就是传销。

### ⚠️ 新型传销的识别要点

| 包装概念 | 实际本质 |
|---------|---------|
| 数字藏品、文创盲盒 | 无实物虚拟标的，击鼓传花 |
| 社交电商、积分商城 | 变相收取入门费 |
| 虚拟种植（种茶、种树、养牛） | 资金盘 |
| "1040阳光工程" | 纯资本运作 |
| 虚拟货币/"矿机" | 价格由平台操控 |

### ⚠️ 法律提醒

- 根据中国法律规定，**参与传销属于违法行为**，投入的资金不受法律保护，一律没收不予返还。
- 组织、领导传销活动罪：可处**5年以上有期徒刑**。
- 2026年1月1日施行的新《治安管理处罚法》首次将组织、领导传销活动纳入行政处罚，可拘留5–15天。

### ⚠️ 防传销原则

- **凡是承诺"一年回本、三年暴富"的项目，直接拒绝**
- **凡是让你发展下线才能赚钱的，都是传销**
- **凡是先交钱才能加入的，都要警惕**
- **不轻信"成功学""国学包装"的投资项目**
- 发现传销活动，请拨打**110**或向当地公安机关举报

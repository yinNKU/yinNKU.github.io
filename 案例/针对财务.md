## 真实案例

### 案例1：北京 — 会计点击"纳税邮件"被植入木马，被骗200万（2024–2026年追回）

会计陈秉收到冒充"区税务稽查"的邮件，点击链接后木马病毒被植入电脑。骗子远程监控其电脑**两周**，摸清其与老板（亲姐）的沟通习惯后，删除真实老板微信，用相同头像昵称的假账号冒充老板下达转账指令。要求转400余万元，实际转出**200万元**。北京石景山警方历时两年追回103万元。

### 案例2：南京 — 假老板群聊潜伏半月，骗走270万（2025年）

财务主管张女士被拉入三人微信群，领导和同事均为冒充。骗子通过远程控制软件在公司电脑潜伏**近半个月**，窃取公司人员架构、业务往来、老板说话风格等信息，选择老板出差时实施诈骗。要求转账270万元，继续转350万时被识破。警方应急联动银行，**270万元全数追回**。

### 案例3：长沙 — 假老板拉群要求150万，财务转出100万（2026年1月）

诈骗分子冒充公司老板，拉群让财务支付"货款"，头像与真老板完全一致。财务支付100万元后接到老板电话发现被骗。长沙警方**追回全部款项**。

### 案例4：苏州 — 老板电脑被远程操控，财务被骗转150万（2025年）

老板电脑被远程操控，财务被拉入微信群，按"老板"指示转账。诈骗分子通过木马病毒操控老板电脑端微信。**被骗150万元**。警方紧急止付，冻结126万，后续追回剩余24万，**全部追回**。

### 案例5：温州龙湾 — 老板微信号被盗，财务被骗转100万（2026年5月）

公司电脑点击可疑"报价文件"中木马病毒，老板微信号被盗。骗子用老板账号向财务发送转账指令，先转**100万元**，要求再转38万元时真老板回公司，识破骗局。警方已立案侦查。

### 案例6：苏州工业园区 — 假钉钉群300万差点转出（2025年10月）

财务收到钉钉群邀请，群内有"老板"和"合作方"，要求立即转账**300万元**。行政部门负责人想起社区民警反诈宣传，致电咨询。民警及时劝阻，**零损失**。

### 案例7：江西定南 — 会计被假老板群骗走近30万（2025年）

会计小王被拉入微信群，群内"老板"头像和昵称与真老板一致，安排其统计公司余额并转账。**被骗29.2万元**。法院以掩饰、隐瞒犯罪所得罪判处被告人覃某有期徒刑一年二个月。

## 一般模式

```
第一步：投毒
  │  发送冒充税务/发票/报价的邮件
  │  或在办公文件/聊天中植入木马
  │  → 远程控制公司电脑
  ▼
第二步：潜伏
  │  监控老板和财务的聊天习惯、公司架构
  │  记录常用语气、业务往来对象
  │  潜伏时间：数天到数周
  ▼
第三步：克隆
  │  删除真实老板的微信/QQ/钉钉
  │  用相同头像、昵称的假账号替代
  │  或拉群冒充"老板+同事+客户"
  ▼
第四步：收割
  │  选择老板出差/开会/无法核实时
  │  强调"急用""先付后补手续"
  │  催促施压，不给思考时间
  │  → 大额转账到指定账户
```

## 受众

- **企业财务人员、会计、出纳**
- 中小型企业（财务制度不完善，老板一人说了算）
- 使用微信/QQ/钉钉等即时通讯办公的企业

---

## 警示

### ⚠️ 财务人员必守的5条铁律

1. **大额转账必须电话或当面核实**——绝不单凭微信/QQ/钉钉指令操作。
2. **不点击陌生邮件链接**——尤其是冒充税务、发票、报价等内容。
3. **定期查杀电脑病毒**——注意电脑异常闪烁、鼠标自行移动等可疑现象。
4. **建立财务分级审批制度**——大额转账需要至少两人确认。
5. **对"紧急转账""先付后补"保持高度警惕**——这是骗子最常用的话术。

### ⚠️ 企业老板要注意

- 电脑和手机定期查杀木马病毒
- 财务人员要有独立核实渠道
- 建立规范的财务制度和审批流程
- 即使再忙，大额转账也要亲自确认

### ⚠️ 发现被骗后

- **立即拨打110报警**，同时联系银行申请紧急止付
- 保留所有聊天记录、转账凭证
- 越快报警，追回的可能性越大

## 真实案例

### 案例1：杭州 — 三倍退款钓鱼网站（2025年7月）

杭州西湖区周女士接到自称电商平台客服的电话，对方准确报出其姓名、订单编号、商品名称，称其购买的化妆品化学成分超标可办理**三倍退款**。周女士信以为真，点击对方发来的钓鱼链接（页面高度仿冒官网），填写了银行卡号、身份证号、支付密码和验证码，资金瞬间被转走，**损失5万元**。

### 案例2：天津滨海 — 快递"理赔"骗局（2026年2月）

居民林某某接到冒充快递公司客服的电话，对方准确报出其在微信小程序购买调味品的订单信息，谎称商品运输中损坏，承诺全额理赔。随后推送虚假"银联客服"电话，诱导下载非官方直播类软件，通过语音操控获取**银行卡信息和验证码**，**损失近2万元**。

### 案例3：广州 — 屏幕共享盗刷（2025年）

广州小陈接到自称"某宝客服"电话，称其快递丢失可双倍赔付。对方引导其下载指定App并开启**屏幕共享**。骗子通过屏幕共享实时看到银行账户信息和验证码，几分钟内转空银行卡，**损失5.8万元**。

### 案例4：海宁 — 30元理赔反失5400元（2026年2月）

海宁长安马老伯（老年群体）接到骗子电话，以"快递小哥出事故"为由，称可办理**30元理赔**。对方诱导其登录"线上服务中心"，索要姓名、身份证号、验证码，并配合人脸识别，最终银行卡被悉数转走，**损失5400余元**。

### 案例5：北京 — 发错货→刷单复合陷阱（2026年1月）

北京张女士在直播间花7元购买搅蒜机，收到坏货后联系售后。对方态度极好，提出加微信"实时监测补发"，随后拉入"商家群"诱导**刷单**。前期小额返利，后期以"操作错误"为由要求连续转账，**损失5万余元**。这是"退款"与"刷单"结合的复合型诈骗。

### 案例6：本溪 — 远程控制App骗局（2026年3月）

张某某接到自称电商客服的电话，称其订单因系统故障被冻结，需点击链接办理"退款理赔"。对方诱导其下载**远程控制App**，先后向所谓的"安全账户"转账5笔，**损失28.6万元**。

### 案例7：宣城 — 网商贷退款骗局

倪先生接到自称客服的电话，称其购买的口红有质量问题需退款。对方诱导其通过支付宝"网商贷"贷款，分3次贷款4万余元进入银行卡。骗子获取验证码后登录其支付宝，将钱全部转走，**损失4万余元**。

## 一般模式

```
第一步：精准信息获取
  │  通过非法渠道购买网购订单信息
  │  → 准确报出姓名、商品、订单号、付款时间
  │  → 降低受害者防备心理
  ▼
第二步：话术施压
  │  常见借口：
  │  · 商品质量问题召回，三倍退款
  │  · 快递丢失，双倍赔付
  │  · 误开通会员/百万保障，不关闭将扣费
  │  · 机票退改签
  │  → 制造利益诱惑 or 恐慌情绪
  ▼
第三步：诱导脱离官方平台
  │  发钓鱼链接（仿冒官网）
  │  诱导下载非官方App
  │  要求添加微信/QQ
  │  要求开启"屏幕共享"或"远程控制"
  │  → **脱离平台监管 = 危险信号**
  ▼
第四步：资金转移
  │  方式一：钓鱼网站套取银行卡号+密码+验证码 → 直接盗刷
  │  方式二：屏幕共享实时监控验证码 → 远程转账
  │  方式三：诱导从借呗/网商贷等平台借款 → 转入"安全账户"
  │  → 资金通过"跑分"账户秒级洗白，追回极难
```

## 受众

- **所有网购人群**（几乎人人都是潜在目标）
- 近期有真实网购订单的人（骗子利用真实订单信息增加可信度）
- **中老年人**（对退款流程不熟悉，易被远程操控）
- 经常使用支付宝、微信支付的用户

## 警示

### ⚠️ 守住四条红线

| 红线 | 具体内容 |
|------|----------|
| ❌ **不轻信精准信息** | 能报出订单信息 ≠ 官方客服，你的信息已泄露是常态 |
| ❌ **不点击不明链接** | 任何短信、聊天中的链接都不要直接点，通过官方App核实 |
| ❌ **不开启屏幕共享** | 凡要求"远程协助退款"开启屏幕共享的，一律挂断报警 |
| ❌ **不透露验证码/密码** | 验证码 = 动态密码，任何正规平台不会索取 |

### ⚠️ 核心认知

- **退、赔款都是原路返回**，不需要点击链接、输入卡号、扫码、提供短信验证码
- **支付宝备用金是个人借款额度**，不是对方的退款——这个骗局利用了人们对陌生概念的盲区
- **不要相信"需要借款提高信用分才能退款"**——这是让你帮忙借钱给骗子

### ⚠️ 新变种警惕（2025–2026）

1. **"发错货"引流到刷单**：商家故意发坏货→主动联系售后→加微信→拉群刷单
2. **"百万保障"骗局**：冒充抖音/微信/快手客服，谎称误开通付费会员或"百万保障"，不关闭将扣费
3. **远程控制App**：不再是简单的钓鱼链接，而是引导下载TeamViewer等远程控制软件，直接操控手机

### ⚠️ 遇到可疑客服怎么办

1. **挂断电话**
2. **通过官方渠道核实**——打开官方App或官网联系客服，不要用对方提供的号码
3. **拨打96110**（全国反诈专线）咨询
4. 已转账的：**立即拨打110报警**

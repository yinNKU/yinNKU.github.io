## 真实案例

### 案例1：宁夏"杀猪盘2.0" — 父母牵线+黄金骗局（2026年）

骗子不直接联系受害者，而是先联系其父母，假扮"相亲对象家长"，由父母转介绍给子女。骗子伪装成"黄金集团中层干部"，以境外出差、内部黄金套利为由，诱导受害者购买实物黄金并快递到指定地址。景女士被骗数十万，兰州康女士被骗 **85万余元**。这是"杀猪盘2.0"——信任链从父母开始建立。

### 案例2：上海"邻居高管" — 68万元被骗光（2025–2026年）

骗子以"邻居房屋漏水"为由加微信，谎称互联网企业高管。每天嘘寒问暖、寄小礼物，甚至寄来一部"专线手机"。诱导下载小众聊天软件和虚假投资App，先让受害者代管账户看到收益上涨，诱使其投入全部积蓄。受害者曾点"剁椒鱼头"外卖到骗子所说公司验证，骗子竟雇佣跑腿员取餐拍照蒙骗过关。钱女士**被骗68万元**。

### 案例3：安徽颍上"精英男" — 20天骗走117万（2026年6月）

骗子伪装成"系统数据总监"，通过社交平台建立恋爱关系。声称有"内部消息"诱导投资，安排"车手"线下取现。短短20天内骗取 **117万元**。

### 案例4：甘肃礼县"假军人" — 团伙涉案超千万（2026年1月）

骗子在"探探"上冒充军人，每日问候建立恋爱关系。诱导下载虚假"天然气交易中心"App，初期小额返利，后续以"缴税""追加投资"为由层层加码。王女士被骗 **38万元**，警方打掉该团伙抓获10人，涉案金额**超1000万元**。

### 案例5：江西"美女主播" — 同时交往20名男子（2025年）

女性犯罪嫌疑人邓某接受话术培训，在交友App上冒充单身女性与约20名男子建立恋爱关系，以"刷礼物证明真心"为由诱导充值打赏，从中分成30%–35%。受害者李杰被骗**近4万元**。

## 一般模式

```
第一步：寻猪（选目标）
  │  社交平台/婚恋网站/相亲群寻找单身、有经济基础的人
  │  精准筛选：年龄、收入、情感状态
  ▼
第二步：养猪（建立信任）
  │  打造完美人设（高管/军人/博士/富二代）
  │  日常嘘寒问暖、寄送礼物
  │  甚至攻克父母长辈，由长辈转介绍（杀猪盘2.0）
  │  → 建立恋爱关系或深度友情
  ▼
第三步：喂猪（展示收益）
  │  让受害者代管"投资账户"，看到高额收益
  │  或声称有"内部消息/漏洞"可套利
  │  初期允许小额提现 → 完全放松警惕
  ▼
第四步：杀猪（收割）
  │  诱导投入全部积蓄 → 平台无法提现
  │  以"缴税""解冻费""保证金"为由继续要钱
  │  安排"车手"线下取现/取黄金
  │  → 拉黑失联，资金秒级洗白
```

## 受众

- **单身、离异、大龄未婚男女**（情感需求强烈）
- 有一定经济基础的30–55岁人群
- 活跃于婚恋网站、社交App（探探、陌陌、小红书等）的用户
- **独居老人**（杀猪盘2.0通过父母下手）

## 警示

1. **凡是网恋 + 推荐投资 = 100%诈骗**——不存在例外。
2. **完美人设不可信**——高富帅/白富美/军人/博士主动倒贴，必有问题。
3. **代管账户看到高收益是假数据**——骗子后台随意操控的数字而已。
4. **不购买实物黄金邮寄给陌生人**——黄金无标识、难溯源，是洗钱首选。
5. **"车手"线下取现是新型洗钱方式**——遇到要求现金/黄金面对面交付的，立即报警。
6. **身份验证可能被造假**——对方收到你点的外卖、寄的礼物，也可能是雇佣跑腿或同伙配合。
7. **不要以为"我没钱就不会被骗"**——骗子会诱导你网贷、刷信用卡、向亲友借钱。

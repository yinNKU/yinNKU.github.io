## 真实案例

### 案例1：山东济宁 — 被骗12.93万元（2025年12月）

济宁网民高某某通过交友软件结识一名异性网友，对方以"增进感情"为由诱导其裸聊，并发送暗藏恶意程序的链接。对方窃取其通讯录和裸聊视频后，以曝光相威胁，高某某被迫多次转账，**累计被骗12.93万元**。警方最终将嫌疑人李某抓获归案。

### 案例2：陕西神木 — "翻墙"裸聊被骗1.5万元（2025年10月）

小帅（化名）在推特收到"美女"私信，对方主动提出裸聊。小帅点击对方发来的链接下载App后视频裸聊，对方随即发来其裸聊视频和通讯录名单，威胁群发。小帅被对方诱导下载视频会议App并开启**"共享屏幕"**，**累计转出1.5万元**。

### 案例3：四川内江 — 5万元网贷险打水漂（2026年2月）

刘先生深夜通过好友验证，对方名为"雅妍"，诱导其下载不知名聊天软件并发私密照片。随后对方发来裸聊视频截图和通讯录，威胁转账。刘先生准备网贷5万元转账，幸被警方及时劝阻，**成功止损5万元**。

### 案例4：浙江嘉兴 — "杀猪盘"+"裸聊"复合诈骗（2025年12月）

邹女士结识一名自称"科研人员"的陌生人，被诱导裸聊后，对方谎称已花86万元"购买公安局副局长职务"，还差16万元"尾款"。邹女士拒绝后，对方用裸聊截图威胁曝光。邹女士最终向警方咨询，**成功止损1.5万元**。警方指出这是 **"杀猪盘"与裸聊敲诈相结合**的复合型犯罪。

### 案例5：全国性犯罪链条 — 95名被害人（2026年1月）

在校大学生张某刚收到"美女"好友申请，裸聊后被勒索**转账7000元**。此人只是全国20余省份95名被害人的缩影。有受害者被迫**转账29次，累计金额近30万元**。犯罪集团在缅甸组建"色播"公司，形成 **"吸粉引流→偷渡→敲诈勒索→洗钱"** 的完整产业链。

### 案例6：江西宁都 — 捣毁3个裸聊诈骗窝点（2026年2月宣判）

严某、曾某、饶某为首的团伙，招聘男性操盘手冒充美女，通过裸聊敲诈勒索。警方在2025年"冬季行动"中捣毁3个窝点，抓获16人，**涉案金额超100万元，受害人上百名**。主犯严某被判处**有期徒刑13年6个月**，罚金18万元。

## 一般模式

```
第一步：引流
  │  社交平台（QQ/微信/陌陌）主动添加好友
  │  低俗网页/涉黄视频App投放广告
  │  境外平台（如推特）主动私信
  │  公共厕所张贴二维码
  ▼
第二步：诱导下载木马App
  │  以"视频聊天需要专用软件"为由
  │  发送链接 → 下载非官方应用商店的App
  │  → **窃取手机通讯录、通话记录、短信、相册**
  ▼
第三步：录制裸聊视频
  │  事先录好的"美女"视频引诱受害者露脸
  │  诱导受害者做出不雅行为并录制
  │  **所谓的"美女"几乎100%是男性操盘手**
  ▼
第四步：威胁勒索钱财
  │  发来裸聊视频截图 + 通讯录名单
  │  话术："你要脸，我要钱，你看着办！"
  │  "没钱可以网贷，钱到位视频删除。"
  │  → 反复加码，直到榨干最后一分钱
```

## 受众

- **以男性为主**，年龄集中在18–45岁
- 活跃于社交软件、交友平台的人群
- 浏览色情网站或使用境外社交平台（推特、Telegram等）的用户
- **深夜时段高发**，孤单感强、警惕性下降时易中招

## 警示

### ⚠️ 事前防范 — 三个"坚决不"

| 行为 | 正确做法 |
|------|----------|
| **陌生链接** | **坚决不点击** |
| **陌生App** | **坚决不下载**（尤其是非官方应用商店的） |
| **陌生人要求裸聊** | **坚决拒绝，立即拉黑** |

### ⚠️ 事中应对 — 发现被骗后

> **千万不要转账！不要转账！不要转账！**

据警方统计，诈骗分子会根据受害者的承受能力**不断加码勒索，直到榨干最后一分钱**。转账后所谓的"删除视频"完全是谎言。

### ⚠️ 事后处理 — 3步止损法

1. **立即停止一切转账行为**
2. **第一时间拨打110报警**（或96110反诈专线）
3. **保存好证据**：聊天记录截图、对方链接/二维码、转账记录、涉诈App名称

### ⚠️ 长期防护

- ✅ 开启手机**应用安装权限验证**
- ✅ 安装**国家反诈中心App**，开启来电预警
- ✅ **关闭"允许安装未知来源应用"** 的开关

### ⚠️ 心理警示

- **不要抱有"花钱消灾"的幻想**——受害者被迫转账29次、累计29万元的案例充分说明：骗子永远不会满足。
- **不要因羞耻或恐惧而沉默**——越早报警，挽回损失的可能性越大。
- **不要相信"删除视频"的承诺**——对方只会变本加厉，这是无底洞。

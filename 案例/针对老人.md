## 真实案例

### 案例1：广州特大保健品诈骗 — 120人被抓，涉案500万（2026年6月破获）

广州荔湾警方组织近400名警力，捣毁7个诈骗窝点，抓获120名嫌疑人，刑拘108人，**涉案金额500余万元**。以"免费健康讲座""免费体检""赠送鸡蛋米面"等为诱饵吸引老人。"专家"义诊把脉，虚构老人身体存在严重问题，制造健康焦虑。高价推销所谓的"神药"如"乌梢蛇牛磺酸冻干粉"，**主要成分实为麦芽糊精**，成本极低，却被以**数百倍高价**售卖。

- **79岁郭阿姨**：被"专家"把脉后吓唬"再不调理要出大事"，先后花**4万元**购买产品
- **76岁王伯**：独居老人，销售人员上门陪聊天、做家务获取信任，花**6万多元**购买号称能治糖尿病、防癌的"神药"，吃了半年毫无效果

### 案例2：四川攀枝花 — 会销保健品诈骗86万（2025年11月二审宣判）

横跨多市，全省涉案店铺9家，**涉案金额86万余元**。手法三步走：

1. **小利引流**：以"免费领鸡蛋""低价粮油"吸引老人进店，带入里屋听"养生课"
2. **温情破冰**：组织老人包饺子、吃火锅，日常嘘寒问暖、上门家访，套取健康状况和经济实力，将客户按收入分为A/B/C三类精准推销
3. **天价收割**：会销现场每5分钟互动一次、安排"托"带头鼓掌。售价2000多元的产品，进价仅200–300元，**利润高达10倍**

15名被告人获刑**10个月至10年**不等，责令退赔。

### 案例3：最高人民法院典型案例 — "二氢槲皮素"抗癌神药（2025年发布）

倪某荣等人以免费年夜饭、赠送小礼品为诱饵，专车接送老人到郊区农家乐。冒充专家虚构"二氢槲皮素"具有抗癌奇效，现场砍价、安排"经销商"哄抢。**涉案金额62万余元**，倪某荣被判**有期徒刑10年3个月**。

### 案例4：AI变声冒充亲人 — 新技术诈骗（2025年）

犯罪分子利用**AI变声技术**模拟"孙子"声音，打电话称打架伤人需赔钱。利用老年人对亲属的关心，编造紧急情况骗取钱财。标志着诈骗进入"深度伪造"新阶段。

### 案例5：短视频培训诈骗 — 2495名老人被骗1309万（2025年）

廖某等人成立公司，以"短视频平台流量扶持""带货培训"为幌子，利用大数据精准筛选"老年人""退休"等关键词。**受害人达2495人，涉案金额1309余万元**。主犯被判**有期徒刑14年**。

### 案例6：上海 — 八旬阿婆被"孙子"电话骗走2万（2025年12月）

80岁王阿婆接到"孙子"电话，称打架伤人需要3万元赔偿，随后自称"所长外甥女"的女子上门取现金。骗子精准掌握了老人子女的出行信息。事后"孙子"再次索要5万元，儿子发现异常报警。警方抓获3名犯罪嫌疑人，追回2万元。

## 一般模式

```
第一步：小利引流
  │  "免费领鸡蛋/大米/食用油"
  │  "免费健康讲座/体检/旅游"
  │  → 吸引老人到场
  ▼
第二步：温情破冰
  │  销售人员嘴甜"爷爷/奶奶"叫个不停
  │  上门陪聊天、做家务、送小礼物
  │  甚至认干亲，比子女还热情
  │  → 建立情感依赖
  ▼
第三步：制造焦虑
  │  "专家"义诊把脉 → 夸大病情
  │  "再不调理要出大事""会癌变"
  │  → 制造健康恐慌
  ▼
第四步：天价收割
  │  成本几十元的产品卖几千元
  │  会销现场"托"带头抢购
  │  "名额有限""今天特价"催促付款
```

## 受众

- **独居老人**（子女不在身边，缺乏关爱）
- **有退休金/积蓄的老年人**
- **有慢性病/健康焦虑的老人**
- **信息渠道单一、不熟悉互联网的老人**

## 警示

### ⚠️ 给老年人

1. **保健品不能替代药品**——任何宣称"包治百病""根治癌症"的都是谎言
2. **"免费"是最贵的东西**——免费讲座、免费体检、免费旅游后面都是陷阱
3. **不要相信"专家"的现场诊断**——真专家不会在讲座上把脉卖药
4. **买药认准"蓝帽子"标志**——正规保健食品有国食健字批准文号
5. **接到"孙子/子女出事"的电话**——先给子女打电话确认，不要急着转账

### ⚠️ 给子女

1. **多陪伴老人**——骗子比子女更"关心"老人，是因为盯上了老人的钱包
2. **定期普及反诈知识**——告诉父母最新的诈骗手法
3. **关注老人的财务状况**——大额支出要问清楚用途
4. **帮老人安装国家反诈中心App**——开启来电预警
5. **提醒老人：遇到可疑情况先打电话问子女**

### ⚠️ 防骗顺口溜

> **免费讲座是诱饵，高价神药是骗局。**
> **温情关怀背后藏，掏空养老钱袋子。**
> **陌生电话要转账，先找子女来商量。**
> **蓝帽子认证才可靠，包治百病不能信。**

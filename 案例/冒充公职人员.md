## 真实案例

### 案例1：湖北黄石 — 新型复合型，被骗25万还沦为"洗钱工具人"（2026年6月）

70多岁邓某接到自称"外地某市公安局民警"的电话，对方精准报出其身份证号、家庭住址，并发送伪造的"逮捕令"。骗子要求邓某下载小众App每天签到报备，打入"专属办案群"多人轮番游说。要求邓某购买**2000元以上新手机**（用于隔离反诈预警、切断亲友联系），诱导其先后转入**25万元养老积蓄**。更诡异的是：钱款转入另一名受害人杨某的账户——杨某此前同样被骗，手机被远程操控，在不知情中沦为"电诈工具人"，形成"连环洗钱链"。警方全额追回25万元。

### 案例2：武汉12岁女孩 — 深夜被"反诈民警"遥控转账1万（2026年4月）

一名12岁女孩小李添加陌生网友后，对方自称"某外省反诈中心民警"，发来带警徽的"警官证"图片，称其账号涉嫌诈骗风险。要求女孩在安静无人的房间通话，指导她拿走母亲的手机、绑定银行卡、修改支付密码，最终转出1万余元。母亲及时发现报警，追回1.1万余元。

### 案例3：湖南益阳 — 诱导购买金条305万线下交付（2025年7月）

诈骗团伙冒充政法系统工作人员联系周某某，谎称其涉嫌诈骗，要求案件保密否则"有杀身之祸"。逐步套取资产信息后，诱导周某某购买**3900克实体金条（价值305万余元）**，谎称金条含"芯片"需交给"公安人员"技术检测。设置接头暗号"0678"，由台湾籍被告人冒充检察院工作人员取走全部金条。

### 案例4：上海 — 冒充检察官诈骗酒店前台（2025–2026年）

无业男子岩某在酒店入住时自称"检察官"，承诺帮前台小霏"内部推荐"检察院书记员工作。一人分饰两角（自己和"Y主任"），通过微信"面试"，以出差缺钱、单位欠薪等理由骗取1.6万余元。小霏为此辞去工作。法院以诈骗罪判处岩某有期徒刑九个月。

### 案例5：上海虹口 — 冒充熟人+公检法+线下取现"三合一"（2025年12月）

80岁王阿婆接到"孙子"电话，称打架伤人需要3万元赔偿，随后自称"所长外甥女"的女子上门取现金。事后"孙子"再次索要5万元，儿子发现异常报警。警方抓获3名犯罪嫌疑人，追回2万元。

## 一般模式

```
第一步：身份冒充
  │  冒充公安/检察院/法院工作人员
  │  发送伪造警官证、逮捕令、通缉令
  │  精准报出身份证号、家庭住址（信息泄露）
  ▼
第二步：制造恐慌
  │  称你涉嫌洗钱/诈骗/非法出入境
  │  要求"案件保密"，不得告知任何人
  │  要求去无人场所，切断与外界联系
  ▼
第三步：隔离监控
  │  要求购买新手机（隔离反诈预警）
  │  要求下载小众App每日签到（建立掌控）
  │  组建"专属办案群"多人轮番洗脑
  ▼
第四步：资金转移
  │  要求转账到"安全账户"核查
  │  或要求购买黄金线下交付
  │  或通过屏幕共享远程操控转账
  │  → 甚至让受害者沦为下一环的"洗钱工具人"
```

## 受众

- **中老年人**（对公检法权威有敬畏心理，信息获取渠道单一）
- **青少年**（社会经验不足，易被恐吓）
- **留学生**（冒充使领馆+公检法复合型）

## 警示

### ⚠️ 核心认知

1. **公检法机关不会通过电话办案**——更不会通过微信/QQ发送逮捕令。
2. **没有所谓的"安全账户"**——要求转账到安全账户的，100%是诈骗。
3. **公检法不会要求你购买新手机隔离联系**——这是为了切断反诈预警。
4. **公检法不会要求你购买黄金/现金线下交付**——这是新型洗钱手法。

### ⚠️ 识别冒充公检法的5个信号

| 信号 | 说明 |
|------|------|
| 🔴 来电显示+00开头或境外号码 | 公检法不会用境外号码办案 |
| 🔴 要求去无人房间/宾馆开房 | 为了切断你与外界的联系 |
| 🔴 发送"警官证""逮捕令"照片 | 这些都是伪造的，真警察不会网上发证 |
| 🔴 要求下载小众App/共享屏幕 | 为了远程监控你的手机操作 |
| 🔴 要求买新手机用于"案件沟通" | 为了隔离反诈预警App |

### ⚠️ 遇到怎么办

1. **立即挂断**，不要与对方多说
2. **拨打110核实**——告诉接线员情况
3. 已转账的：**立即拨打110报警**，越快越好
4. 提醒家人：尤其是独居老人，要定期普及反诈知识

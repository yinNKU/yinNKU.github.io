冒充熟人诈骗

冒充熟人冒充领导猜猜我是谁冒充晚辈上门取现AI变声同学借钱
凡是涉及钱财的熟人电话/信息，务必通过其他渠道核实——直接打电话给本人或家属，不要用对方提供的号码。

真实案例

案例1：山东临沂 — 昔日同窗三年诓骗16万（2023–2026年）

大学舍友陈某以"生活困难""银行卡被封需解冻费"等借口多次借款。2025年4月，陈某谎称父亲有"人脉资源"，可托关系为受害人安排工作，索要"活动经费"数万元。后又编造"突发重病""摔倒流产需切除输卵管""父亲车祸去世"等极端谎言。受害人累计被骗16万余元，钱款全部用于直播打赏。套路：老同学→托关系安排工作→苦情戏→冒充家人通话。

案例2：深圳 — 水果摊主冒充警察和领导家属（2024–2025年）

水果摊贩贝某冒充"民警"，称"岳父是大学党委书记"，虚构升任"治安科科长""派出所副所长"等身份，以"介绍实验室项目"为名，索要"见面费""辛苦费"等，骗取受害人10万余元。以招摇撞骗罪判处有期徒刑十一个月。

案例3：山西临汾 — 同事"救急"诈骗16万（2025年）

快餐店员工乔某利用同事关系，以"家人撞死人要赔钱""朋友做生意缺资金""家里有人患急病"等虚假理由反复借款，甚至冒用丈夫名义谎称自己已去世逃避还款，累计骗取同事王某16万余元，全部用于偿还个人债务和网络刷单。以诈骗罪判处有期徒刑三年，缓刑四年，并处罚金二万元。

案例4：北京 — 冒充孙子打电话，上门取走6万（2026年1月）

骗子冒充儿子打电话给84岁老人，称"旅游开车撞了人需要赔钱"，派人上门取走6万元现金，又要求再取8万元，被银行和警方拦截。骗子精准掌握了老人子女的出行信息。此类案件从打电话到上门取现最快20分钟。

案例5：上海 — 冒充"邻居"加好友，网恋投资诈骗68万（2025年）

骗子以"房屋漏水"为由错加邻居好友，自称互联网企业高管，通过寄送礼物建立好感。受害人甚至用"点剁椒鱼头外卖"试探，骗子雇佣跑腿员取货拍照蒙骗。之后诱导下载小众聊天软件和虚假投资平台，骗光68万元积蓄。

一般模式

模式一：冒充晚辈（针对老人） 冒充子女/孙子/孙女打电话 → "出车祸/打架被抓/嫖娼被罚需要赔钱" → 要求取现金 → 派人上门取钱 → 利用老人对子女的关心，快速行动 模式二：冒充领导/熟人 "猜猜我是谁" → 冒充领导/老朋友 → "明天来我办公室一趟"（见面之前） → "正在陪领导，需要转账帮忙" → 要求转账到指定账户 模式三：冒充同学/同事 利用旧情或日常接触 → 编造紧急变故 → 小额多次借款 → 最终失联 模式四：冒充邻居/新朋友 以"漏水""加错人"为由建立联系 → 打造精英人设 → 寄送礼物建立信任 → 诱导投资/借款

受众

  • 独居老人（冒充晚辈类，利用亲情关怀）
  • 中年人（冒充领导/朋友类，利用社会关系）
  • 同事/同学圈（冒充熟人借款类，利用日常信任）

警示

  1. 凡是涉及钱财的熟人电话/信息，务必通过其他渠道核实——直接打电话给本人或家属，不要用对方提供的号码。
  2. 独居老人是冒充晚辈诈骗的高危人群——子女要多向老人普及反诈知识，提醒"凡是打电话要钱的，先给子女打电话确认"。
  3. "托关系、走后门安排工作"基本是诈骗——正规就业不需要私下转账。
  4. 看到00开头、+号开头的境外来电要警惕——极有可能是诈骗电话。
  5. "上门取现金"是冒充熟人诈骗的新趋势——正规办事不会派人上门取现金。
  6. 不要碍于情面不核实——即使是熟人的账号，涉及转账也要电话确认。
  7. 发现被骗及时保存聊天记录和转账凭证，立即报警。

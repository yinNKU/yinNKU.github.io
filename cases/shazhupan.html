一般模式

第一步：寻猪（选目标） │ 社交平台/婚恋网站/相亲群寻找单身、有经济基础的人 │ 精准筛选：年龄、收入、情感状态 ▼ 第二步：养猪（建立信任） │ 打造完美人设（高管/军人/博士/富二代） │ 日常嘘寒问暖、寄送礼物 │ 甚至攻克父母长辈，由长辈转介绍（杀猪盘2.0） │ → 建立恋爱关系或深度友情 ▼ 第三步：喂猪（展示收益） │ 让受害者代管"投资账户"，看到高额收益 │ 或声称有"内部消息/漏洞"可套利 │ 初期允许小额提现 → 完全放松警惕 ▼ 第四步：杀猪（收割） │ 诱导投入全部积蓄 → 平台无法提现 │ 以"缴税""解冻费""保证金"为由继续要钱 │ 安排"车手"线下取现/取黄金 │ → 拉黑失联，资金秒级洗白

  • 有一定经济基础的30–55岁人群
  • 活跃于婚恋网站、社交App（探探、陌陌、小红书等）的用户

警示

  1. 凡是网恋 + 推荐投资 = 100%诈骗——不存在例外。
  2. 完美人设不可信——高富帅/白富美/军人/博士主动倒贴，必有问题。
  3. 代管账户看到高收益是假数据——骗子后台随意操控的数字而已。
  4. 不购买实物黄金邮寄给陌生人——黄金无标识、难溯源，是洗钱首选。
  5. "车手"线下取现是新型洗钱方式——遇到要求现金/黄金面对面交付的，立即报警。
  6. 身份验证可能被造假——对方收到你点的外卖、寄的礼物，也可能是雇佣跑腿或同伙配合。
  7. 不要以为"我没钱就不会被骗"——骗子会诱导你网贷、刷信用卡、向亲友借钱。

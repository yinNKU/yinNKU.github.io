虚假招工诈骗

虚假招聘招转培黑中介内推保录无薪试岗代办证件培训贷
凡是入职前先让交钱的，一律拒绝——正规招聘不会收取任何费用。

真实案例

案例1：北京"招转培" — 面试变培训，诱导贷款1.88万（2025年9月）

求职者古某接到北京某传媒科技公司面试邀请，面试运营助理岗位。公司口头承诺月薪8000元但未签劳动合同，随后以"入职需经过实训"为由，诱导古某签订《新媒体运营实训咨询服务协议》，收取实训费1.88万元。因古某无力支付，公司诱导其在小贷平台办理分期贷款，每月需还1777元。北京朝阳区多部门联合处置，查清以孙某为首的高薪引流→诱导培训→借贷施骗全链条诈骗团伙。

案例2：苏州"虚假高薪" — 时薪26元直招实为层层转包（2024–2025年）

求职者张强在某招聘网站看到迈泰景人力资源公司发布"某知名电子厂直招、时薪26元"的招聘信息。实际该公司无职业中介资质，冒用招聘网站账号发布虚假信息。张强被层层转介至第三家人力资源机构，被收取被褥费280元和车费100元，实际薪资与宣传严重不符。涉事公司被没收全部违法所得1.2万元，处以5万元罚款。

案例3：上海 — 残疾男子找工作三年被骗300万（2022–2025年）

被害人徐先生身患残疾，其妻子的亲戚陈某谎称有某优质公司的内部入职资源。陈某伪造知名高校毕业证、学籍证明、优秀毕业生证书等材料，还注册两个微信小号假扮公司高管。3年间以资质办理、岗前培训、体检、医保代缴等名目持续收费，累计骗取300余万元（含徐先生夫妇及岳父岳母的养老积蓄），全部用于网络主播打赏和个人挥霍。2026年5月，检察机关以诈骗罪提起公诉。

案例4：吉林"国企内推" — 号称保录国企骗取财物（2025年7月）

某人力资源公司在网络平台发布"高薪保录""直签国企"等招聘信息，向求职者暗示"有关系""送钱就能入职"，骗取巨额财物。经查，相关国有企业并未委托该公司招聘。人社部门责令改正、退还费用、停止职业中介活动。

案例5：浙江玉环"代办证件" — 建筑行业招工群诈骗（2024–2025年）

嫌疑人吴某某在建筑行业招工微信群发布虚假招工信息，以"代办特种设备作业证""证件迁转"为由，收取300至600元不等的费用，收款后立即拉黑求职者。2024年4月至2025年5月期间，骗取5名被害人共计3803元。吴某某被判处有期徒刑六个月，并处罚金3000元。

一般模式

模式一：招转培（最常见） 高薪招聘 → 面试称能力不足需培训 → 推荐"合作培训机构" → 诱导贷款交培训费 → 培训结束后不兑现承诺的岗位 模式二：黑中介收费 发布虚假高薪信息 → 收取报名费/体检费/被褥费 → 层层转介到更差的岗位 → 或直接拉黑 模式三：内推保录 号称有关系、有内部渠道 → 索要"活动经费" → 伪造录用通知 → 迟迟无法入职 → 最终失联 模式四：熟人诈骗 利用亲友信任 → 虚构入职渠道 → 以各种名目长期收费 → 直到被揭穿

受众

  • 应届毕业生（求职心切，缺乏经验）
  • 待业人员、农民工（信息不对称，急于找工作）
  • 想进国企/事业单位的人（被"内推保录"吸引）
  • 残疾人等弱势求职群体（更容易被利用）

警示

  1. 凡是入职前先让交钱的，一律拒绝——正规招聘不会收取任何费用。
  2. 警惕"招转培"——面试变成培训推荐，还让你贷款交学费，一定是骗局。
  3. "低门槛、高薪酬"是危险信号——没有学历经验要求却承诺高薪，基本是骗局。
  4. 核实公司资质——通过"国家企业信用信息公示系统"查询公司是否真实存在。
  5. 不相信"国企内推""内部名额"——正规国企招聘公开透明，不存在花钱买路。
  6. 警惕熟人介绍的"特殊门路"——求职就业有正规流程，不存在内部特批。
  7. 保留证据——聊天记录、合同协议、转账凭证，遇到问题及时向人社部门投诉或拨打110报警。

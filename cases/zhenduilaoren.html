案例1：广州特大保健品诈骗 — 120人被抓，涉案500万（2026年6月破获）

广州荔湾警方组织近400名警力，捣毁7个诈骗窝点，抓获120名嫌疑人，刑拘108人，涉案金额500余万元。以"免费健康讲座""免费体检""赠送鸡蛋米面"等为诱饵吸引老人。"专家"义诊把脉，虚构老人身体存在严重问题，制造健康焦虑。高价推销所谓的"神药"如"乌梢蛇牛磺酸冻干粉"，主要成分实为麦芽糊精，成本极低，却被以数百倍高价售卖。

  • 79岁郭阿姨：被"专家"把脉后吓唬"再不调理要出大事"，先后花4万元购买产品
  • 76岁王伯：独居老人，销售人员上门陪聊天、做家务获取信任，花6万多元购买号称能治糖尿病、防癌的"神药"，吃了半年毫无效果

案例2：四川攀枝花 — 会销保健品诈骗86万（2025年11月二审宣判）

横跨多市，全省涉案店铺9家，涉案金额86万余元。手法三步走：1.小利引流——"免费领鸡蛋""低价粮油"吸引老人进店；2.温情破冰——组织包饺子、吃火锅，日常嘘寒问暖、上门家访，套取健康状况和经济实力，将客户按收入分为A/B/C三类精准推销；3.天价收割——会销现场每5分钟互动一次、安排"托"带头鼓掌，售价2000多元的产品进价仅200–300元，利润高达10倍。15名被告人获刑10个月至10年不等，责令退赔。

案例3：最高人民法院典型案例 — "二氢槲皮素"抗癌神药（2025年发布）

倪某荣等人以免费年夜饭、赠送小礼品为诱饵，专车接送老人到郊区农家乐。冒充专家虚构"二氢槲皮素"具有抗癌奇效，现场砍价、安排"经销商"哄抢。涉案金额62万余元，倪某荣被判有期徒刑10年3个月。

案例4：AI变声冒充亲人 — 新技术诈骗（2025年）

案例5：短视频培训诈骗 — 2495名老人被骗1309万（2025年）

一般模式

第一步：小利引流 │ "免费领鸡蛋/大米/食用油" │ "免费健康讲座/体检/旅游" │ → 吸引老人到场 ▼ 第二步：温情破冰 │ 销售人员嘴甜"爷爷/奶奶"叫个不停 │ 上门陪聊天、做家务、送小礼物 │ 甚至认干亲，比子女还热情 │ → 建立情感依赖 ▼ 第三步：制造焦虑 │ "专家"义诊把脉 → 夸大病情 │ "再不调理要出大事""会癌变" │ → 制造健康恐慌 ▼ 第四步：天价收割 │ 成本几十元的产品卖几千元 │ 会销现场"托"带头抢购 │ "名额有限""今天特价"催促付款

受众

  • 独居老人（子女不在身边，缺乏关爱）
  • 有退休金/积蓄的老年人
  • 有慢性病/健康焦虑的老人
  • 信息渠道单一、不熟悉互联网的老人

一般模式

第一步：精准信息获取 │ 通过非法渠道购买网购订单信息 │ → 准确报出姓名、商品、订单号、付款时间 → 降低防备 ▼ 第二步：话术施压 │ 常见借口：商品质量问题召回三倍退款 / 快递丢失双倍赔付 │ 误开通会员/百万保障不关闭将扣费 / 机票退改签 ▼ 第三步：诱导脱离官方平台 │ 发钓鱼链接（仿冒官网）/ 诱导下载非官方App │ 要求添加微信/QQ / 要求开启"屏幕共享"或"远程控制" │ → 脱离平台监管 = 危险信号 ▼ 第四步：资金转移 │ 方式一：钓鱼网站套取银行卡号+密码+验证码 → 直接盗刷 │ 方式二：屏幕共享实时监控验证码 → 远程转账 │ 方式三：诱导从借呗/网商贷等平台借款 → 转入"安全账户"

受众

  • 所有网购人群（几乎人人都是潜在目标）
  • 近期有真实网购订单的人（骗子利用真实订单信息增加可信度）
  • 中老年人（对退款流程不熟悉，易被远程操控）

警示

⚠️ 守住四条红线

❌ 不开启屏幕共享凡要求"远程协助退款"开启屏幕共享的，一律挂断报警
